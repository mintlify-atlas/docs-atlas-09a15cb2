## The AI Coding Agent Ecosystem

Welcome to **Awesome OpenCode** — a curated collection of plugins, themes, agents, projects, and resources for [OpenCode](https://opencode.ai/), the AI coding agent for the terminal built by [Anomaly](https://github.com/anomalyco).

<Note>
This is a **community-driven** directory with **58 plugins**, **4 agents**, **29 projects**, and comprehensive coverage of the OpenCode ecosystem. Whether you're looking to extend functionality, customize your experience, or discover what's possible with OpenCode, you're in the right place.
</Note>

## Popular Highlights

### Power User Plugins

- **Oh My Opencode** - Background agents, pre-built LSP/AST/MCP tools, and curated agents
- **Micode** - Structured Brainstorm-Plan-Implement workflow with session continuity
- **Opencode Workspace** - Bundled multi-agent orchestration with 16 components
- **Opencode Mem** - Persistent memory with vector database for long-term context

### Developer Experience

- **Opencode Worktree** - Zero-friction git worktrees with auto-spawned terminals
- **Devcontainers** - Multi-branch devcontainer instances with auto-assigned ports
- **Background Agents** - Claude Code-style async delegation and context persistence
- **Smart Title** - Auto-generate meaningful session titles using AI

### Project Management

- **OpenChamber** - Web and desktop GUI with VS Code extension
- **Agent of Empires** - Multi-session TUI with tmux and Docker sandboxing
- **Vibe Kanban** - Kanban board for orchestrating AI agents in parallel
- **OpenWork** - Desktop GUI alternative to Claude Cowork
